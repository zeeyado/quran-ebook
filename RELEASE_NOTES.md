### New since v0.5.0

**41 languages** (was 25) — added Persian, Malay, Portuguese, Somali, Hausa, Swahili, Bosnian, Chinese, Italian, Hindi, Tamil, Korean, Japanese, Albanian, Pashto, Dutch, Azerbaijani, Uzbek, Tajik, Kazakh, Kurdish, Uyghur, Malayalam, Thai, Vietnamese, Tagalog, Amharic, Yoruba, Swedish, Polish, Ukrainian, Norwegian.

**The Clear Quran** — Dr. Mustafa Khattab translation in four variants: plain text (bilingual + interactive) and annotated edition with 1,270 footnotes extracted from the ebook.

**Vertical spacing fix** — ayah number markers (`line-height:1`) no longer inflate line boxes, reducing the vertical gap variance between lines in Arabic script, so pages are much more uniform. 

**Quran StarDict dictionary** (English, optional) — word-by-word dictionary for KOReader's built-in lookup. 22,000+ entries with translation, transliteration, morphological analysis (root, lemma, part of speech), and Lane's Lexicon root definitions. Work in progress. Only for QPC Uthmani Hafs encoding.

**Validation fixes** — XML escaping for translator names with special characters and non-Latin angle brackets in translation text.
